This package contains an implementation of the RECOMP method - a technique for detecting regions of recombination among a set of sequences. The method is described in the following papers:

  • D. Ruths and L. Nakhleh. "RECOMP: A Parsimony-Based Method for Detecting Recombination." In Proc. of 4th Asia Pacific Bioinformatics Conference, 2006.
  • D. Ruths and L. Nakhleh. "Recombination and Phylogeny: Effects and Detection." International Journal on Bioinformatics Research and Applications, 1(2):202-212, 2005.
